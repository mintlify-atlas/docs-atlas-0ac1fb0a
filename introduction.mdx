## What is yt-dlp-ejs?

yt-dlp-ejs is a specialized library that provides external JavaScript execution capabilities for [yt-dlp](https://github.com/yt-dlp/yt-dlp), the popular video downloader. It solves YouTube's signature and n-parameter challenges by preprocessing and executing JavaScript code in a sandboxed environment.

The library is designed to work seamlessly across multiple JavaScript runtimes, making it flexible and reliable for different deployment scenarios.

## How It Works

yt-dlp-ejs preprocesses YouTube player JavaScript code to extract signature and n-parameter solving functions. These functions are then made available through a clean Python API that yt-dlp can use to decode video URLs.

The library uses:
- **meriyah** for parsing JavaScript into an Abstract Syntax Tree (AST)
- **astring** for generating JavaScript code from the AST
- **rollup** for bundling the solver code with dependencies
